nested_index()

nested_index() : callable

Returns a callable which returns an item from a nested array corresponding to the given path.

Examples:

$array = [
    'foo' => [
        'bar' => [
            'baz' => 42
        ]
    ]
];

$getIndexFooBar = fn\nested_index('foo', 'bar');
$getIndexFooBarBaz = fn\nested_index('foo', 'bar', 'baz');

$getIndexFooBar($array)
=> ['baz' => 42]

$getIndexFooBarBaz($array)
=> 42

…()

Dies if ever called, a stub for unimplemented code.

The "…" method functions similarly to Perl6's "..." operator (the "yada, yada, yada" operator) and dies if it's ever called. It's used as a stub for unwritten code.
